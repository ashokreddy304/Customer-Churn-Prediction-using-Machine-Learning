## 🚀 Project Overview
Customer churn is a major challenge for subscription-based businesses.  
This project predicts whether a customer will **Churn (Leave)** or **Stay**, allowing companies to take proactive retention actions.

---

## 🛠 Tech Stack
| Component | Technology |
|-----------|------------|
| Programming | Python |
| Libraries | Pandas, NumPy, Scikit-learn, LightGBM, Matplotlib, Seaborn |
| Model Type | Classification |
| Feature Selection | Random Forest Importance |

---

## 📊 Model Performance

| Model | Accuracy |
|--------|---------|
| **LGBM Classifier** | **83%** ✅ |
| XGBoost | 82% |
| Random Forest | 82% |
| Logistic Regression | 81% |
| Naive Bayes | 80% |
| KNN | 76% |
| SVM | 60% |

✅ **Final Model:** LightGBM (Best Accuracy)

---

## 🎯 Key Features
✔ Performed EDA, preprocessing, encoding, and feature selection  
✔ Selected **Top 10 features using Random Forest**  
✔ Trained multiple ML models & compared results  
✔ Final LGBM model predicts **Churn or Not Churn** for new customers  
✔ Supports **YES/NO output** with **probability score**
